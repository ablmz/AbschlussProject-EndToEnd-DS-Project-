Textuelle und Sterne Vorhersage mithilfe Machinelles Lernen Modelle


Maschinelles Lernen ist ein Oberbegriff für die „künstliche“ Generierung von Wissen aus Erfahrung: Ein künstliches System lernt aus Beispielen und kann diese nach Beendigung der Lernphase verallgemeinern. Dazu bauen Algorithmen beim maschinellen Lernen ein statistisches Modell auf, das auf Trainingsdaten beruht. Das heißt, es werden nicht einfach die Beispiele auswendig gelernt, sondern Muster und Gesetzmäßigkeiten in den Lerndaten erkannt. So kann das System auch unbekannte Daten beurteilen (Lerntransfer).

In dieser Webseite können Sie Zwei verschiedene ML Modelergebnisse, und zwar Sternevorhersage, Erfahrungsvorhersage erleben. Bei der der Sternevorhersage kann unser erstes Model die eingegebene Texte als Sterne bewerten. Bei der Erfahrungsvorhersage kann unser zweites Model die eingegebene Texte als gut oder schlecht bewerten.
